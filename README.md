# Clone-of-amazon
### Amazon-clone project.
It adds search functionality, hover effects, dynamic “See More” clicks, Back-to-Top behavior, and a fake cart counter.
| Feature                                | Description                                             |
| -------------------------------------- | ------------------------------------------------------- |
| ✔ Search Function                      | Alerts the text you search for ("Search ROHIT" working) |
| ✔ Hover Animations                     | Border highlight effects become smooth and interactive  |
| ✔ "See More" Buttons                   | Alerts category name when clicked                       |
| ✔ Fake Cart Counter                    | Shows “Cart (1)”, “Cart (2)”, etc                       |
| ✔ Back to Top                          | Smooth scroll back to top                               |
| ✔ Clickable Panel (Today's deal, etc.) | Alerts which menu option you clicked                    |
| ✔ Language Button Demo                 | Shows message                                           |
